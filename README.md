## 🛠️ Running Locally

Clone the repo:

```bash
git clone https://github.com/Ven-Sib/Portfolio_new.git
cd Portfolio_new
npm install
npm run dev
````

```bash
npm run build
npm run deploy
```

> Automatically builds and deploys to [https://Ven-Sib.github.io/Portfolio\_new](https://Ven-Sib.github.io/Portfolio_new)
